1. 변수의 수치와 승률의 상관관계¶

특정 스텟의 수치가 승률에 얼마나 관계가 있는지 피어슨r 상관계수를 통해서 알아본다.

상관관계를 구하는 기본적인 cor() 식은 각 데이터(점)에 해당하는 빈도수를 고려하지 않고 계산되기 때문에 현재 분석에 알맞지 않다.

따라서 기존 피어슨r 상관계수 공식을 아래와 같이 변형시켰다.

참고 : https://ko.wikipedia.org/wiki/%ED%94%BC%EC%96%B4%EC%8A%A8_%EC%83%81%EA%B4%80_%EA%B3%84%EC%88%98

In [15]:
# 각 점마다 가중치 z[i](해당 점의 빈도 수) 를 주는 방식
# x : 변수의 수치
# y : 변수에 따른 승률
# z : x, y값에 해당하는 빈도수

correlation  <- function(x,y,z){
    # 변수의 평균 x_mean을 구하기 위해 sum0 과 sum2 를 선언
    # 변수에 따른 승률 평균 y_mean 을 구하기 위해 sum1 을 선언 
    # 변형된 피어슨 r을 구하기 위해 sum3,sum4,sum5,x_mean, y_mean 을 선언
    # 합을 구하기 전에 sum0 부터 sum5 까지 0으로 초기화. 
    sum0  <- 0    # 각 변수의 수치의 점의 빈도 수 x 변수의 수치의 합
    sum1  <- 0    # 각 변수의 수치의 변수에 따른 승률 x 변수의 수치의 합
    sum2  <- 0    # 총 경기 수
    sum3  <- 0    # x와 y의 공분산
    sum4  <- 0    # x의 표준편차
    sum5  <- 0    # y의 표준편차 
    
    # 반복문을 통해 sum0,sum1,sum2 를 구하기
    for (i in 1:length(x)){
        sum0 = sum0 + z[i]*x[i]
        sum1 = sum1 + z[i]*y[i]
        sum2 = sum2 + z[i]
    }
    
    # 변수의 평균과 변수에 따른 승률 평균 구하기 
    x_mean  <- sum0/sum2
    y_mean  <- sum1/sum2
    
    # 반복문을 통해 sum3,sum4,sum5 구하기
    for (i in 1:length(x)){
        sum3 = sum3 + z[i]*((x[i]-x_mean)*(y[i]-y_mean))
        sum4 = sum4 + z[i]*((x[i]-x_mean)^2)
        sum5 = sum5 + z[i]*((y[i]-y_mean)^2)
        
    }
    
    # 변형된 피어슨r 구하기
    r  <- sum3/sqrt(sum4*sum5)
    return (r)
}

  • 각 스텟별 수치에 따른 승리 확률의 상관 도출 가능

2. 분석 결과를 산점도 시각화¶

범주를 양의 상관관계와 음의 상관관계로 나눈후, 상관계수의 절대값이 높은 순서대로 산점도를 그린다.

승리를 많이 한다라는 것의 기준을 승률 50% 이상으로 설정한다.
